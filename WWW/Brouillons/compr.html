Les images compressées

Les 3 formats Web ... lequel choisir pour mon image ?

Pour avoir une page web qui se charge rapidement il faut limiter la quantité et le poids des images : Pour le web : image < 30 ko !!!
Il faut choisir intelligemment le format de compression pour avoir le meilleur rapport qualité / poids ! Il n’y a pas de format idéal, le choix de compression varie selon le contenu de l’image !

  PNG-32   * très bons dégradés : grâce aux 16,7 millions de couleurs.
* Transparence : Permet la transparence.
* Translucidité : Permet les couleurs translucides (avec le canal Alpha).

** => Non-destructeur, contours précis, canal Alpha

 * Alpha non supporté par les anciens navigateurs : Si 1 Alpha 254 alors la couleur translucide est inconnue des anciens navigateurs (Internet Explorer 6) ... Mais si alpha vaut 0 ou 255 (transparence ou opacité) alors c’est bien géré.
** => Parfois plus lourd qu’un autre format

Synthèse : Certains formats sont similaires donc à tester pour comparer la qualité/poids du rendu :

  • JPEG et PNG-24 : pour les photos
  • GIF et PNG-8 : pour les dessins (palette de 256 couleurs)
Les autres formats sont à oublier pour une publication sur un site car ils sont trop lourds. Mais ils peuvent être utilisés comme format de travail.
